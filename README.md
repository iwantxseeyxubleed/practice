# ООО ЭлектроСервис

Проект разделён на несколько файлов, чтобы код было проще проверять и поддерживать.

## Запуск

```powershell
pip install -r requirements.txt
python main.py
```

## Структура проекта

- `main.py` — точка входа в программу.
- `config.py` — пути, цвета, шрифты и общие настройки.
- `database.py` — подключение к SQLite и создание таблиц.
- `import_data.py` — импорт начальных данных из Excel.
- `repository.py` — функции работы с базой данных: авторизация, товары, удаление, сохранение.
- `utils.py` — работа с изображениями.
- `app.py` — главное окно, авторизация, список товаров, поиск, фильтрация, сортировка.
- `product_form.py` — форма добавления и редактирования товара.
- `electroservice.db` — база данных SQLite.
- `import/` — исходные Excel-файлы и изображения.
- `images/` — изображения товаров, которые используются программой.

## Роли

- Гость — просмотр товаров.
- Менеджер — просмотр товаров, поиск, фильтрация и сортировка.
- Администратор — все возможности менеджера, а также добавление, редактирование и удаление товаров.
